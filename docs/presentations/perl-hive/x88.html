5. Hive Server Versus Hive Server 2

Summary: Use Hive Server 2.

  • Supports ODBC via a driver on the Hive 2 server.

  • Supports concurrency in both ODBC and Thrift connections.

  • Better support for authentication and authorization.

  • Better unified support for auditing and logging.

  • Hive Server has memory leaks that may not be fixable.

The analysis above is taken from Carl Steinbach's slide presentation on the subject.
